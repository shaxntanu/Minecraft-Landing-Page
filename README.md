# Minecraft Landing Page Re-imagined

A premium, modern, and responsive redesign of the Minecraft landing page. This project replaces the static HTML/CSS with a robust **TypeScript + Vite** architecture and adds smooth, physics-based animations using **Motion** (vanilla framer-motion).
